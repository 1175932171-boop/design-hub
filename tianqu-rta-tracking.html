0文档说明

ℹ️
本文档基于 天渠 RTA 管理平台 Demo（tianqu-rta-demo.html） 及 PRD v2.0 梳理，覆盖三大模块的全部用户行为事件，包括页面曝光（PV）、点击、弹窗操作、提交、跳转等。

埋点采用 SPM 三段式 命名规范，所有事件上报时携带全局公共参数。数据用于：

  • 核心功能使用率统计（实验创建/策略上线/推全等）
  • 操作转化漏斗分析（弹窗打开→提交完成）
  • 异常行为监控（置灰按钮点击尝试、下线阻断频率）
  • 用户行为热图（各功能点击分布）

✦命名规范

事件 ID 格式

采用 SPM 三段式：rta_{module}_{action}_{object}

段位含义枚举值
rta产品前缀（固定）rta
module模块rtaid / exp / strat / global
action行为类型click / view / submit / search / confirm / cancel
object操作对象具体操作名，如 add_btn / edit_modal / offline_confirm 等

事件类型说明

类型说明示例
PV页面或弹窗曝光模块切换至实验管理、弹窗打开
CLICK用户点击行为点击编辑、点击版本标签、点击策略名称
SUBMIT表单提交/确认类确认上线、提交扩桶申请、保存策略
EXPOSE按钮/模块曝光（区分可用/置灰）上线按钮置灰曝光、下线按钮置灰曝光
