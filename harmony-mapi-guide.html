① 查询鸿蒙应用列表接口
GET /open_api/v3.0/tool/harmony_app/get/
字段名类型必填说明
advertiser_id integer 必填 广告主账号 ID
app_name string 选填 鸿蒙应用名称，用于模糊查询筛选
package_name string 选填 鸿蒙应用包名（如 com.example.myapp）
page / page_size integer 选填 分页参数，page_size 最大 100
返回字段 app_id 即为鸿蒙应用在巨量平台的唯一标识，后续创建项目时使用。
② 创建项目（Project）接口 v3.0
POST /open_api/v3.0/project/create/
字段名类型枚举值 / 说明备注
landing_type enum APP LINK MICRO_GAME ... 推广目标。投放鸿蒙 APP 下载固定填 APP
app_type enum HARMONYOS ANDROID IOS 鸿蒙关键字段 投放鸿蒙 APP 时填 HARMONYOS；系统会在项目层绑定华为应用市场的应用
app_id string 鸿蒙应用在巨量平台的 app_id 通过「查询鸿蒙应用列表」接口获取
download_url string 华为应用市场下载链接 鸿蒙应用市场落地页 URL，如 https://appgallery.huawei.com/app/...
app_promotion_type enum DOWNLOAD LAUNCH 子目标：DOWNLOAD = 应用下载；LAUNCH = 应用调起。鸿蒙目前主要用 DOWNLOAD
track_url string 第三方监测链接（点击监测） 鸿蒙 APP 点击归因监测链接，支持 Adjust / AppsFlyer / 自建等
注意：当 app_type = HARMONYOS 时，以下安卓专属功能不可设置：深度转化归因渠道包、安卓定向「应用安装状态」等。
③ 事件管理（转化跟踪）关键字段
字段名枚举值说明
asset_type HARMONY / APP 资产类型，鸿蒙应用选 HARMONY
attribution_type CLICK 目前鸿蒙支持点击归因，后续可能开放融合归因
app_api_conversion_types ACTIVATE（激活） REGISTER（注册） PURCHASE（付费） ... 鸿蒙支持的回传事件类型与 Android 基本一致
④ 定向参数（受众定向）
鸿蒙应用投放时，系统会自动将受众圈定在持有鸿蒙设备的用户。如需进一步精细化定向，可配合以下字段：
字段名枚举值说明
device_type PHONE PAD 设备类型定向
os_type（audience 下） HARMONYOS ANDROID IOS 操作系统定向，可指定仅投鸿蒙设备

① 广告单元创建接口
POST /rest/openapi/v2/ad_unit/create
字段名类型枚举值（鸿蒙相关）备注
app_type enum HARMONYOS ANDROID IOS 鸿蒙关键字段 应用平台类型，鸿蒙 APP 下载选 HARMONYOS
scene_id enum APP_DOWNLOAD 营销目标/场景，APP 下载场景
download_url string 华为应用市场 AppGallery 链接 鸿蒙 APP 的华为应用市场下载链接，必填
app_id string 快手平台应用 ID（鸿蒙应用预先注册） 需在快手广告后台「资产 → 应用管理」中提前注册鸿蒙应用并获取
package_name string 鸿蒙应用包名 如 com.example.myapp
click_track_url string 第三方点击监测链接 鸿蒙 APP 归因监测链接
impression_track_url string 第三方曝光监测链接 可选
② 广告单元更新接口
POST /rest/openapi/v2/ad_unit/update

字段结构与创建接口一致，通过 unit_id 指定要更新的广告单元。

③ 广告计划（Campaign）层关键设置
POST /rest/openapi/v2/campaign/create
字段名枚举值说明

④ 转化归因配置
快手鸿蒙 APP 投放需在快手广告后台完成：应用注册（HarmonyOS）→ 转化事件配置 → 联调验证，再通过 MAPI 引用已联调的 conversion_id。
字段名枚举值/说明备注
conversion_id integer，后台获取 已联调的鸿蒙转化 ID，在广告单元中引用
deep_conversion_type RETENTION（次留） IN_APP_ACTION（深度行为） 鸿蒙 APP 支持深度转化目标（根据开放进度）

⚠️ 通用注意事项
1. 应用市场链接：鸿蒙 APP 下载链接须为华为 AppGallery 应用市场链接，格式通常为：
https://appgallery.huawei.com/app/C<appid>
2. OAID 归因：鸿蒙 NEXT 系统不支持 IMEI/AndroidID，归因设备标识使用 OAID（鸿蒙设备广告 ID），接入各平台 SDK 时需适配鸿蒙 OAID 获取方式。
3. 深度转化：部分平台（如巨量引擎）鸿蒙 APP 暂不支持某些安卓专属深度转化能力，以实际 MAPI 错误提示为准。
4. 转化联调：所有平台均要求在正式上线前完成鸿蒙 APP 转化数据联调，需使用搭载鸿蒙 NEXT（HarmonyOS 5.0+）的华为/荣耀设备进行验证。
5. 流量规模：目前各平台鸿蒙流量体量仍在增长阶段，建议同时保留 Android 计划跑量，鸿蒙单独建计划便于分析效果。
6. 回传方式：各平台鸿蒙投放均主要支持「应用 API 回传」方式，即广告主服务端主动上报转化事件，不依赖 SDK 自动上报。
本文档基于各平台公开 MAPI 文档整理，字段枚举以官方最新文档为准。建议定期查阅各平台开放平台更新日志。
